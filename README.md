# 北京浮生记：中年优化

一个 60 天文字策略生存游戏 MVP。玩家扮演 39 岁被优化的北京互联网中层，在房贷、家庭、求职、社保和现金流之间撑过两个月。

## 运行

```bash
npm install
npm run dev
```

打开终端输出的本地地址即可游玩。

## 已实现

- React + TypeScript + Vite 单机 Web 版
- localStorage 存档、读档、重开
- 8 个北京地点
- 24 个可选行动
- 每日机会市场：倒卖、外包快单、内推窗口、父母复查、社保窗口等限时机会牌
- 机会牌支持过期、风险等级、随机结果和连锁事件
- 外包/咨询支持定金、应收尾款和概率到账
- 14 天现金流雷区，提前显示房贷、月中账单、尾款、社保和父母健康风险
- 每天 3 个行动点
- 34 张数据驱动事件卡
- 第 3 天房贷扣款
- 第 60 天餐桌事件和结局判定
- 9 个阶段性结局
