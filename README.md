# 北京浮生记：中年优化

一个 60 天文字策略生存游戏 MVP。玩家扮演 39 岁被优化的北京互联网中层，在房贷、家庭、求职、社保和现金流之间撑过两个月。

## 运行

```bash
npm install
npm run dev
```

打开终端输出的本地地址即可游玩。

## 已实现

- React + TypeScript + Vite 单机 Web 版
- localStorage 存档、读档、重开
- 8 个北京地点
- 24 个可选行动
- 每天 3 个行动点
- 34 张数据驱动事件卡
- 第 3 天房贷扣款
- 第 60 天餐桌事件和结局判定
- 9 个阶段性结局

## 说明

游戏中的人物、公司、金额、房贷、工资、房价与事件均为虚构和戏剧化调参，不代表现实数据或法律、金融建议。
